# Periodontal Staging and Grading Diagnostic Tool

This is a periodontal staging and grading diagnostic tool based on the 2018 International New Classification of Periodontal and Peri-implant Diseases. This tool aims to help dental professionals and students quickly and accurately perform periodontal staging and grading assessments.

## Project Goals

- Provide a simple, intuitive interface to help users understand the periodontal staging and grading system
- Automatically determine periodontal staging and grading based on clinical parameters (such as CAL, bone loss, PD, etc.)
- Visualize the diagnostic process to help users understand the assessment criteria
- Provide educational functionality to help users learn the periodontal staging and grading system

## Technology Stack

This project is built using simple HTML, CSS, and JavaScript, requiring no additional environments or components, and can run directly in a browser.

## Usage Instructions

1. Open the `index.html` file
2. Enter relevant clinical parameters
3. Get periodontal staging and grading diagnostic results
4. View the diagnostic flowchart to understand the assessment criteria

## Project Structure

- `index.html`: Main page
- `css/`: Style files
- `js/`: JavaScript scripts
- `images/`: Icons and image resources
