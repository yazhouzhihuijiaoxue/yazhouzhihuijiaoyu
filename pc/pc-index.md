# 教师端首页产品文档

## 页面概述
教师端首页是教师进入牙周智慧教学平台后的主工作台，提供教学数据概览、快速操作入口和实时活动监控。

## 功能目标与业务流程

### 核心功能
1. **教学数据概览**：实时展示学生总数、课程数量、待处理任务等关键指标
2. **快速操作导航**：提供作业管理、批改、成绩录入等核心功能的快速入口
3. **数据可视化分析**：通过图表展示学生成绩分布和作业提交趋势
4. **实时活动监控**：显示学生最近的学习活动动态

### 业务目的
- **教学效率提升**：通过数据概览帮助教师快速了解教学状态
- **任务优先级管理**：突出显示待处理任务，引导教师高效工作
- **教学质量监控**：通过数据图表辅助教师进行教学效果评估

### 业务流程
教师登录系统后，首先看到首页的数据概览，根据待处理任务数量决定下一步操作方向，通过快速入口进入具体功能模块。

## 交互逻辑与状态变化

### 用户操作 -> 系统响应
- **点击"作业管理"卡片** -> 跳转到作业与课程管理页面 (pc-homework-manage.html)
- **点击"作业批改"卡片** -> 跳转到作业批改页面 (pc-homework-review.html)
- **点击"成绩录入"卡片** -> 跳转到成绩录入页面 (pc-score-input.html)
- **点击"学术大厅"卡片** -> 跳转到知识整合中心页面 (pc-knowledge-center.html)
- **页面加载时** -> 自动初始化并显示两个图表（学生成绩分布、作业提交趋势）

### 页面元素状态变化
- **快速操作卡片**：鼠标悬停时卡片上浮2px，阴影加深，视觉上产生点击反馈
- **数据统计卡片**：数值以醒目的32px大字体显示，主色调强调
- **图表区域**：加载完成后显示交互式图表，支持鼠标悬停查看详细数据
- **最近活动列表**：按时间倒序排列，最新活动显示在最上方

## 各种状态参数规则

### URL参数控制
- **无参数状态**：显示默认首页内容，包含所有功能模块
- **?tab=stats**：直接定位到数据统计区域（预留功能）
- **?tab=activity**：直接定位到最近活动区域（预留功能）

### 页面状态导向
- **待处理任务数量 > 0**：在欢迎区域显示提醒信息，引导教师优先处理
- **无待处理任务**：欢迎区域显示鼓励性提示，如"所有任务已完成，继续保持！"

## 表单字段规则

### 页面无输入表单
教师端首页为纯展示和导航页面，不包含用户输入表单元素。所有交互通过点击导航卡片实现页面跳转功能。
