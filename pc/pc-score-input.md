# 成绩录入产品文档

## 页面概述
成绩录入页面是教师录入牙周智慧教学平台虚拟仿真机和仿人头模训练成绩的核心界面，支持批量录入、数据统计和成绩分布分析。

## 功能目标与业务流程

### 核心功能
1. **虚拟仿真机成绩录入**：录入学生在虚拟仿真系统中的操作规范、技术熟练度和时间效率得分
2. **仿人头模成绩录入**：录入学生在仿人头模训练中的实际操作成绩（功能开发中）
3. **批量成绩录入**：支持为多个学生同时录入相同的成绩
4. **成绩数据统计**：实时显示平均分、最高分、最低分等统计指标
5. **成绩分布分析**：通过图表展示学生成绩的分布情况

### 业务目的
- **成绩标准化管理**：通过统一的评分标准确保成绩录入的规范性和公平性
- **教学效果评估**：通过成绩统计分析帮助教师了解教学效果和学生掌握程度
- **工作效率提升**：通过批量录入功能减少重复操作，提高成绩录入效率

### 业务流程
教师进入页面后选择虚拟仿真机成绩录入标签，选择课程和训练项目，系统加载对应学生列表。教师可以逐个录入学生成绩，也可以使用批量录入功能。录入过程中系统实时计算总分和统计指标，录入完成后提交所有成绩到系统。

## 交互逻辑与状态变化

### 用户操作 -> 系统响应
- **选择"虚拟仿真机成绩录入"标签** -> 显示虚拟仿真成绩录入界面
- **选择"仿人头模成绩录入"标签** -> 显示仿人头模成绩录入界面（功能开发中）
- **从课程下拉菜单选择课程** -> 系统加载该课程的学生列表
- **选择训练项目** -> 更新评分标准和学生列表（预留功能）
- **设置考核日期** -> 记录成绩对应的考核时间
- **在评分标准区域设置分值** -> 系统实时计算总分并显示
- **在学生成绩输入框输入分数** -> 系统实时计算该学生的总分并更新状态
- **点击"批量录入成绩"按钮** -> 弹出批量录入模态框
- **在批量录入模态框中设置分数** -> 系统验证分数有效性
- **点击"应用到所有学生"按钮** -> 将所有待录入学生的成绩设置为批量分数
- **点击"从系统导入"按钮** -> 尝试从虚拟仿真系统自动导入成绩（预留功能）
- **点击"提交所有成绩"按钮** -> 验证所有学生成绩已录入后提交到系统

### 页面元素状态变化
- **标签页切换**：当前激活的标签页底部显示蓝色下划线，文字变为强调色
- **学生成绩状态标签**：已录入成绩显示绿色"已录入"标签，待录入显示黄色"待录入"标签
- **总分计算**：当操作规范、技术熟练、时间效率任一分数变化时，自动重新计算总分
- **批量录入模态框**：显示时背景变暗，模态框居中显示，包含三个分数输入框
- **统计卡片**：随着成绩录入实时更新平均分、最高分、最低分等统计指标
- **成绩分布图表**：学生成绩变化时自动重新渲染分布图表

## 各种状态参数规则

### URL参数控制
- **无参数状态**：默认显示虚拟仿真机成绩录入标签页
- **?tab=headmodel**：直接显示仿人头模成绩录入标签页
- **?course=1**：默认选择ID为1的课程并加载学生列表
- **?training=basic-operation**：默认选择基础操作训练项目
- **?date=2026-03-09**：默认设置考核日期为2026年3月9日

### 页面状态导向
- **未选择课程**：显示"请选择课程开始录入成绩"的空状态提示
- **所有学生成绩已录入**："提交所有成绩"按钮变为可点击的绿色状态
- **存在待录入学生**："提交所有成绩"按钮保持灰色不可点击状态
- **批量录入应用后**：所有待录入学生的成绩被设置为批量分数，状态变为已录入
- **从系统导入成功**：自动填充学生成绩，状态变为已录入

## 表单字段规则

### 成绩录入设置表单
- **选择课程**：必填项，从下拉菜单中选择已创建的课程，选项包括"牙周基础理论"、"临床操作技能"
- **训练项目**：必填项，可选值为"basic-operation"（基础操作训练）、"advanced-technique"（高级技术训练）、"emergency-handling"（应急处理训练）
- **考核日期**：必填项，必须选择过去或今天的日期，格式为YYYY-MM-DD

### 评分标准设置
- **操作规范分**：数值范围0-40，必须为整数，默认值0
- **技术熟练分**：数值范围0-40，必须为整数，默认值0  
- **时间效率分**：数值范围0-20，必须为整数，默认值0
- **总分**：自动计算（操作规范分+技术熟练分+时间效率分），只读显示

### 学生成绩录入
- **操作规范分**：数值范围0-40，必须为整数，输入超出范围时自动修正为边界值
- **技术熟练分**：数值范围0-40，必须为整数，输入超出范围时自动修正为边界值
- **时间效率分**：数值范围0-20，必须为整数，输入超出范围时自动修正为边界值
- **总分**：自动计算，范围0-100，只读显示
- **成绩状态**：自动判断，总分>0时为"已录入"，否则为"待录入"

### 批量录入规则
- **批量分数应用范围**：只应用到状态为"待录入"的学生
- **批量分数验证**：批量设置的分数同样需要符合各分项的范围限制
- **批量操作确认**：应用前显示确认提示，防止误操作
