# 作业与课程管理产品文档

## 页面概述
作业与课程管理页面是教师进行牙周智慧教学平台教学任务管理的核心界面，支持作业发布、课程安排和教学进度监控。

## 功能目标与业务流程

### 核心功能
1. **作业管理**：创建、编辑、发布和查看作业信息
2. **课程管理**：管理课程信息、学生选课情况和课程进度
3. **作业搜索与筛选**：按状态、关键词快速定位特定作业
4. **教学数据统计**：实时显示作业提交率、学生参与度等关键指标

### 业务目的
- **教学任务组织**：系统化地管理所有教学作业和课程安排
- **教学效率提升**：通过数据统计帮助教师了解教学效果
- **任务优先级管理**：根据作业状态和截止时间合理安排工作

### 业务流程
教师进入页面后，默认显示作业管理标签页，可以通过顶部标签切换作业管理和课程管理。在作业管理中可以发布新作业、搜索现有作业、按状态筛选作业，并查看每个作业的详细统计信息。

## 交互逻辑与状态变化

### 用户操作 -> 系统响应
- **点击"作业管理"标签** -> 切换到作业管理视图，显示所有作业列表
- **点击"课程管理"标签** -> 切换到课程管理视图，显示所有课程信息
- **点击"+ 发布新作业"按钮** -> 弹出发布作业模态框，包含作业标题、课程选择、描述和截止时间输入框
- **在搜索框输入关键词** -> 实时过滤作业列表，只显示标题或描述包含关键词的作业
- **选择状态筛选器** -> 按作业状态（全部/已发布/草稿/已结束）过滤作业列表
- **点击作业卡片上的"批改作业"按钮** -> 跳转到作业批改页面 (pc-homework-review.html)
- **点击作业卡片上的"编辑"按钮** -> 弹出编辑作业信息的模态框（预留功能）
- **点击作业卡片上的"详情"按钮** -> 显示作业的详细信息（预留功能）
- **点击课程卡片上的"课程管理"按钮** -> 跳转到课程详细管理页面（预留功能）

### 页面元素状态变化
- **标签页切换**：当前激活的标签页底部显示蓝色下划线，文字变为强调色
- **作业状态标签**：不同状态的作业显示不同颜色（已发布-蓝色、草稿-灰色、已结束-绿色）
- **作业卡片悬停**：鼠标悬停时卡片阴影加深，产生上浮效果
- **模态框交互**：发布作业模态框显示时，背景变暗，模态框内容居中显示
- **表单验证**：发布作业时，如果必填字段为空，提交按钮保持可点击状态但会提示用户填写完整信息

## 各种状态参数规则

### URL参数控制
- **无参数状态**：默认显示作业管理标签页
- **?tab=course**：直接显示课程管理标签页内容
- **?status=published**：在作业管理标签页中默认筛选已发布的作业
- **?status=draft**：在作业管理标签页中默认筛选草稿状态的作业
- **?status=ended**：在作业管理标签页中默认筛选已结束的作业

### 页面状态导向
- **作业状态为"已发布"**：显示批改作业按钮，教师可以开始批改
- **作业状态为"草稿"**：批改作业按钮置灰不可点击
- **作业状态为"已结束"**：批改作业按钮变为"查看批改结果"
- **作业提交率 < 50%**：在作业统计区域用橙色强调低提交率
- **作业提交率 > 90%**：在作业统计区域用绿色强调高提交率

## 表单字段规则

### 发布作业表单
- **作业标题**：必填项，长度限制1-50个字符，支持中英文、数字和常见标点符号
- **所属课程**：必填项，从下拉菜单中选择已创建的课程，选项包括"牙周基础理论"、"临床操作技能"、"科研拓展课"
- **作业描述**：选填项，长度限制0-500个字符，支持多行文本输入
- **截止时间**：必填项，必须选择未来的日期时间，格式为YYYY-MM-DD HH:MM

### 搜索和筛选
- **搜索关键词**：长度限制0-50个字符，支持中英文和数字搜索
- **状态筛选器**：可选值为""（全部）、"published"（已发布）、"draft"（草稿）、"ended"（已结束）

### 数据统计显示规则
- **学生总数**：显示该作业对应的所有学生数量，数值范围0-999
- **已提交学生数**：显示已提交作业的学生数量，数值范围0-学生总数
- **提交率**：自动计算（已提交学生数/学生总数*100%），保留整数百分比
