# 牙周炎分期分级诊断工具

这是一个基于2018年牙周病和植体周病国际新分类的牙周炎分期分级诊断工具。该工具旨在帮助牙科专业人员和学生快速、准确地进行牙周炎的分期和分级判断。

## 项目目标

- 提供一个简单、直观的界面，帮助用户理解牙周炎分期分级系统
- 根据临床参数（如CAL、骨吸收、PD等）自动判断牙周炎的分期和分级
- 可视化展示诊断流程，帮助用户理解判断依据
- 提供教育功能，帮助用户学习牙周炎分期分级系统

## 技术栈

本项目使用简单的HTML、CSS和JavaScript构建，无需安装额外的环境或组件，可以直接在浏览器中运行。

## 使用方法

1. 打开`index.html`文件
2. 输入相关临床参数
3. 获取牙周炎分期和分级的诊断结果
4. 查看诊断流程图，了解诊断依据

## 项目结构

- `index.html`: 主页面
- `css/`: 样式文件
- `js/`: JavaScript脚本
- `images/`: 图标和图像资源
