# 作业批改产品文档

## 页面概述
作业批改页面是教师对牙周智慧教学平台学生提交的作业进行评分和反馈的核心界面，支持批量批改、进度跟踪和详细评语录入。

## 功能目标与业务流程

### 核心功能
1. **作业选择与切换**：教师可以选择需要批改的作业，系统自动加载对应学生列表
2. **学生作业查看**：查看学生提交的作业内容、附件和提交时间
3. **评分与评语录入**：对作业进行评分（0-100分）并录入详细评语
4. **批改状态管理**：标记作业批改状态（待批改/已批改/需要修改）
5. **批改进度跟踪**：实时显示当前作业的批改进度和完成比例

### 业务目的
- **教学质量保障**：通过标准化评分确保作业评价的公平性和一致性
- **学习反馈及时性**：为学生提供及时的作业反馈，促进学习改进
- **批改效率提升**：通过进度跟踪和快速导航提高批改工作效率

### 业务流程
教师进入页面后首先选择需要批改的作业，系统加载该作业的所有学生提交情况。教师通过左侧学生列表选择学生，查看作业内容后进行评分和评语录入，完成后标记批改状态，系统自动更新批改进度。

## 交互逻辑与状态变化

### 用户操作 -> 系统响应
- **从下拉菜单选择作业** -> 系统加载该作业的学生列表，清空当前选择的学生
- **点击左侧学生列表项** -> 在右侧内容区域显示该学生的作业提交内容和批改表单
- **在得分输入框输入分数** -> 系统实时验证分数有效性（0-100分）
- **选择批改状态** -> 根据选择的状态改变学生列表中的状态显示
- **点击"保存批改"按钮** -> 保存当前学生的批改信息，但不标记为完成
- **点击"完成批改"按钮** -> 保存批改信息并将学生标记为已批改状态
- **点击"上一个"按钮** -> 切换到学生列表中的上一个学生
- **点击"下一个"按钮** -> 切换到学生列表中的下一个学生
- **点击附件名称** -> 弹出查看附件的模态框或下载附件

### 页面元素状态变化
- **学生列表项**：当前选中的学生项显示蓝色左侧边框和浅绿色背景
- **批改进度条**：根据已批改学生比例实时更新进度条长度和百分比显示
- **得分输入框**：输入无效分数（<0或>100）时边框变为红色警告色
- **完成批改按钮**：只有当学生状态为"待批改"时才显示此按钮
- **学生状态标签**：不同状态显示不同颜色（已提交-绿色、未提交-橙色、已批改-蓝色）

## 各种状态参数规则

### URL参数控制
- **无参数状态**：显示作业选择界面，学生列表为空
- **?homework=1**：默认选择ID为1的作业并加载学生列表
- **?student=20260001**：在指定作业中默认选择学号为20260001的学生
- **?tab=submitted**：在学生列表中默认筛选已提交的学生
- **?tab=not-submitted**：在学生列表中默认筛选未提交的学生
- **?tab=reviewed**：在学生列表中默认筛选已批改的学生

### 页面状态导向
- **学生状态为"未提交"**：批改区域显示"该学生尚未提交作业"的提示信息
- **所有学生已批改**：显示"所有作业批改完成"的祝贺信息
- **批改进度达到100%**：进度条显示为完整的绿色，并显示完成提示
- **首次进入页面**：显示"请选择作业开始批改"的引导提示

## 表单字段规则

### 评分表单
- **得分**：必填项，数值范围0-100，支持整数和小数（保留1位小数），输入无效值时显示红色边框警告
- **批改状态**：必填项，可选值为"pending"（待批改）、"reviewed"（已批改）、"need-revision"（需要修改）
- **评语**：选填项，长度限制0-1000个字符，支持多行文本输入，支持中英文、数字和常见标点符号

### 数据验证规则
- **得分验证**：必须为数字，范围0-100，超出范围时提示"请输入有效的分数（0-100）"
- **必填项验证**：得分和批改状态为必填项，保存时如未填写会提示用户
- **附件查看**：点击附件名称时，系统根据附件类型提供预览或下载功能

### 批改状态流转规则
- **待批改 -> 已批改**：教师完成评分和评语后点击"完成批改"按钮
- **待批改 -> 需要修改**：教师认为作业需要学生修改时选择此状态
- **需要修改 -> 已批改**：学生重新提交后，教师重新批改并标记为已批改
- **状态不可逆**：已批改状态不能改回待批改或需要修改状态
